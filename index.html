Adding a Note

3 steps

Drop a new HTML file anywhere one level deep. The two shared files at the repo root (audio-sync.css / audio-sync.js) handle all audio panels and GitHub sync — no copy-paste needed.

  1. 01
    Link the shared stylesheet in <head>
    <link rel="stylesheet" href="../audio-sync.css">
  2. 02
    Place an audio panel placeholder after each section heading
    <section class="segment" id="s1">
  <div class="segment-head">…</div>
  <div class="audio-panel" id="audio-panel-s1"></div>
</section>
  3. 03
    Add the config + script before </body>
    <script>
window.ALAB_PAGE_CONFIG = {
  storageKey: 'alab_audio_data_my_note_v1',
  metaKey:    'alab_audio_meta_my_note_v1',
  topicIds:   ['s1', 's2', 's3'],
  ghPath:     'audio-data/my_note.json'
};
</script>
<script src="../audio-sync.js"></script>
